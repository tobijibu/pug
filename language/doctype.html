Doctype

DOCTYPE

基本は以下の通りです。

doctype html
<!DOCTYPE html>

DOCTYPEショートカット

以下は、よく使われるDOCTYPEのショートカットです。

doctype html

<!DOCTYPE html>

doctype xml

<?xml version="1.0" encoding="utf-8" ?>

doctype transitional

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

doctype strict

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

doctype frameset

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Frameset//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-frameset.dtd">

doctype 1.1

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN" "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">

doctype basic

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML Basic 1.1//EN" "http://www.w3.org/TR/xhtml-basic/xhtml-basic11.dtd">

doctype mobile

<!DOCTYPE html PUBLIC "-//WAPFORUM//DTD XHTML Mobile 1.2//EN" "http://www.openmobilealliance.org/tech/DTD/xhtml-mobile12.dtd">

doctype plist

<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">

カスタムDOCTYPE

独自のDOCTYPEを使うこともできます。

doctype html PUBLIC "-//W3C//DTD XHTML Basic 1.1//EN"
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML Basic 1.1//EN">

DOCTYPEオプション

DOCTYPEは出力にバッファされるだけでなく、コンパイルにも影響が出る場合があります。 例えば、自己終了タグが>か/>かで、HTMLとXMLのどちらに指定されているかで異なります。 また、boolean属性の出力も影響を受ける可能性があります。

何らかの理由でdoctypeを利用できない場合(例えばHTMLを表示するだけ等)に、 doctypeを指定するには、doctypeオプションを使って対応することができます。 

var pug = require('./');
 
var source = 'img(src="foo.png")';
 
pug.render(source);
// => '<img src="foo.png"/>'
 
pug.render(source, {doctype: 'xml'});
// => '<img src="foo.png"></img>'
 
pug.render(source, {doctype: 'html'});
// => '<img src="foo.png">'
